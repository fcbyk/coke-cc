## 01 基础入门

::: details 补全下面代码，交换 `变量a` 和 `变量b` 的值
```cpp
#include <iostream>
using namespace std;

int main() {
    int a, b;
    cin >> a >> b;
    
    // 这里编写交换逻辑

    cout << a << " " << b << endl;
    return 0;
}
```
:::

**示例​：** 
- 输入 **`4 6`** ,  输出  **`6 4`** 
- 输入 **`23 78`** ,  输出  **`78 23`**
