## 2 分析运行结果

观察以下代码，它的运行结果会是怎么样呢？

:::: code-group

```cpp [代码1]
#include <iostream>
using namespace std;

int main() {
    int a = 10.5;
    cout << a ;
    return 0;
}
```

```cpp [代码2]
#include <iostream>
using namespace std;

int main() {
    int a = 5;
    {
        int a = 10;
        cout << a;
    }
    cout << a;
    return 0;
}
```

```cpp [代码3]
#include <iostream>
using namespace std;

int main() {
    bool b1 = true;
    bool b2 = false;
    cout << b1 << " " << b2;
    return 0;
}
```

```cpp [代码4]
#include <iostream>
using namespace std;

int main() {
    double a = 100, b = 200 , c = a + b;
    cout << c;
    return 0;
}
```
