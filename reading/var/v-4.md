## 4 分析运行结果

观察以下代码，它的运行结果会是怎么样呢？

:::: code-group

```cpp [代码1]
#include <iostream>
using namespace std;

int main() {
    int a = 10;
    int b = 20;

    {
        int a = 30;
        cout << a+b << endl;
        {
            int b = 100;
            cout << a+b << endl;
        }
    }

    cout << a+b << endl;
    
    return 0;
}
```

```cpp [代码2]

    int a = 200;
    cout << a << b;
    return 0;
}
```
