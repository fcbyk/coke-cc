## 习题1：盛小通当裁判 <Badge type="tip" text="课堂习题" />
**【描述】**  盛小通当裁判盛小通的同学在做一个游戏，让盛小通当裁判，游戏的规则是，选手要写一个两位数，如果个位数字是十位数字的整数倍，那就说明他通过了。如果通过了就输出 Yes ，否则就输出 No<br/>
**【输入描述】**  输入一个整数 n （ 10 < n < 99 ），表示写下的数字。<br/>
**【输出描述】**  按照规则输出 Yes 或 No <br/>

##### 输入输出示例

<RunningResult>
24 (输入)<br/>
Yes (输出)<br/>
</RunningResult>

##### 示例代码

<PasswordProtected>

```cpp
#include <iostream>
using namespace std;
int main(){
    // 输入一个整数
    int a;
    cin >> a;

    // 获取个位和十位
    int ge = a%10;
    int shi = a/10;

    // 判断是否为整数倍
    if (ge%shi==0){
        cout << "Yes" << endl;
    } else{
        cout << "No" << endl;
    }

    return 0;
}
```
