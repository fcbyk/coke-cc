## 习题2：星期转换 <Badge type="easy" />

##### 题目
编写程序将数字1-7转换为对应的星期名称。

##### 要求
1. 使用 `switch` 语句实现转换
2. 1对应星期一，7对应星期日
3. 输入其他数字显示"输入错误"

##### 输入输出示例
<RunningResult>
请输入星期数字(1-7)：3<br/>
星期三
</RunningResult>

<RunningResult>
请输入星期数字(1-7)：8<br/>
输入错误
</RunningResult>

##### 示例代码
<PasswordProtected>

```cpp
#include <iostream>
using namespace std;

int main() {
    int day;
    
    cout << "请输入星期数字(1-7)：";
    cin >> day;
    
    switch(day) {
        case 1: cout << "星期一"; break;
        case 2: cout << "星期二"; break;
        case 3: cout << "星期三"; break;
        case 4: cout << "星期四"; break;
        case 5: cout << "星期五"; break;
        case 6: cout << "星期六"; break;
        case 7: cout << "星期日"; break;
        default: cout << "输入错误";
    }
    
    return 0;
}
```
