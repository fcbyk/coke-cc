
## 练习4：计算矩形的面积和周长
**【描述】** 编写一个程序，计算并输出矩形的面积和周长。假设矩形的长为 a，宽为 b。<br/>
**【输入描述】** 通过输入，为题目中的 `a`，`b` 设置一个具体的值<br/>
**【输出描述】** 计算面积和周长，并输出结果。<br/>

##### 输入输出示例

<RunningResult>
10 20 (这里为用户输入)<br/>
面积：200<br/>
周长：60
</RunningResult>

##### 示例代码

<PasswordProtected>

```cpp 
#include <iostream>
using namespace std;

int main() {

    // 定义两个变量，用来矩形的长和宽
    double a, b;

    // 输入矩形的长和宽
    cin >> a >> b;

    // 按要求使用cout输出结果
    cout << "面积：" << a*b << endl;
    cout << "周长：" << (a+b)*2 << endl;

    return 0;
}
```
