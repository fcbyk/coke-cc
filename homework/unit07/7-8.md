## 习题8：判断数的正负性 <Badge type="info" text="课后练习" />

**【描述】** 输入一个整数，判断该数是正数、负数还是零 <br/>
**【输入】** 输入一个整数 <br/>
**【输出】** 如果是正数输出`正数`，如果是负数输出`负数`，如果是零输出`零` <br/>
**【示例1】** 输入`5`，输出`正数` <br/>
**【示例2】** 输入`-3`，输出`负数` <br/>
**【示例3】** 输入`0`，输出`零` <br/>

##### 示例代码

<PasswordProtected>

```cpp
#include <iostream>
using namespace std;

int main() {
    int num;
    cin >> num;  // 输入数字
    
    if(num > 0) {
        cout << "正数" << endl;
    } else if(num < 0) {
        cout << "负数" << endl;
    } else {
        cout << "零" << endl;
    }
    
    return 0;
}
```
