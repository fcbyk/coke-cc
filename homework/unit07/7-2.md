## 习题2：if-else 分析题
> 观察以下代码，分析运行结果

::: code-group

```cpp [程序1]
#include <iostream>
using namespace std;

int main() {
    int score = 85;
    
    if (score >= 60) {
        cout << "及格";
    } else {
        cout << "不及格";
    }
    
    return 0;
}
```

```cpp [程序2]
#include <iostream>
using namespace std;

int main() {
    int score = 30;
    
    if (score >= 90) {
        cout << "优秀";
    } else {

        if (score >= 60) {
            cout << "及格";
        } else {
            cout << "不及格";
        }
        
    }
    
    return 0;
}
```

```cpp [程序3]

    return 0;
}
```
