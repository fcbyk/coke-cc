## 05 找最大值

输入三个数，输出最大值

**示例​：** 
- 输入 **`23 55 33`** ,  输出  **`55`** 
- 输入 **`765 453 867`** ,  输出  **`867`** 

#### 参考答案

```cpp
#include <iostream>
using namespace std;

int main() {
    int a, b, c;
    cin >> a >> b >> c;
    
    int max = a;
    if (b > max) max = b;
    if (c > max) max = c;

    cout <<  max << endl;
    
    return 0;
}
```
