## 02 位数之和

输入一个四位数，输出其每位数相加的和

**示例​：** 
- 输入 **`1234`** ,  输出  **`10`** 
- 输入 **`5542`** ,  输出  **`16`**

```cpp [解法1]
#include <iostream>
using namespace std;

int main() {
    int num;
    cin >> num;

    // 分解各位数字（先除后取余）
    int thousand = num / 1000;         // 千位
    int hundred = (num / 100) % 10;   // 百位
    int ten = (num / 10) % 10;    // 十位
    int one = num % 10;           // 个位

    int sum = thousand + hundred + ten + one;
    cout << sum << endl;
    return 0;
}
```
