## 02 无临时变量交换

不使用临时变量，实现两个变量的交换，可复制题目1的代码

**示例​：** 
- 输入 **`23 76`** ,  输出  **`76 23`** 
- 输入 **`545 34`** ,  输出  **`34 545`**

```cpp [算术运算]
#include <iostream>
using namespace std;

int main() {
    int a, b;
    cin >> a >> b;
    
    // 交换逻辑
    a = a + b;
    b = a - b;
    a = a - b;

    cout << a << " " << b << endl;
    return 0;
}
```

```cpp [异或运算]
#include <iostream>
using namespace std;

int main() {
    int a, b;
    cin >> a >> b;
    
    // 交换逻辑
    a = a ^ b;
    b = a ^ b;
    a = a ^ b;

    cout << a << " " << b << endl;
    return 0;
}
```
