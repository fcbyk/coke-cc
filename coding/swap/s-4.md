## 04 封装成函数

::: details 补全以下代码，设计一个函数，实现两个 `int变量` 的交换功能。

```cpp
#include <iostream>
using namespace std;

// 此处编写交换函数

int main() {
    int a, b;
    cin >> a >> b;
    
    // 此处调用交换函数
    
    cout << a << " " << b << endl;
    return 0;
}
```
:::

**示例​：** 
- 输入 **`0 2`** ,  输出  **`2 0`** 
- 输入 **`3 4`** ,  输出  **`4 3`**

```cpp [使用指针]
#include <iostream>
using namespace std;

// 指针版本的交换函数
void swap(int *x, int *y) {
    // 解指针
    int temp = *x;
    *x = *y;
    *y = temp;
}

int main() {
    int a, b;
    cin >> a >> b;
    
    swap(&a, &b); // 传递地址
    
    cout << a << " " << b << endl;
    return 0;
}
```
