## 走进C++ <Badge type="info" text="了解" />

#### <span style="display: inline-block; padding: 4px 8px; background: #E3F2FD; border-left: 4px solid #2196F3; border-radius: 2px;">C++是什么</span>
C++ 是一种编程语言，就像我们用汉语、英语和人交流一样，C++ 是人和计算机交流的工具。它可以让计算机按照我们的指令完成任务，比如显示文字、进行计算或者制作小游戏等。

---

#### <span style="display: inline-block; padding: 4px 8px; background: #E3F2FD; border-left: 4px solid #2196F3; border-radius: 2px;">对比于其他语言，C++的特点</span>
如果把编程语言比作交通工具：  
• **Python（简单）**：像自行车，学起来容易，但速度慢一些。  
• **Scratch（图形化）**：像乐高积木，拖拽就能玩，但只能搭简单造型。  
• **C++（强大）**：像赛车，跑得飞快，能完成复杂任务，但需要学更多“驾驶技巧”！  

--- 

#### <span style="display: inline-block; padding: 4px 8px; background: #E3F2FD; border-left: 4px solid #2196F3; border-radius: 2px;">计算机是如何执行C++的代码的</span>
计算机其实只懂“0和1”组成的机器语言，要让计算机听懂C++，需要一个**编译器**帮忙：  
1. **你写代码**：用C++写下代码，比如让计算机显示“你好！”。  
2. **编译器翻译**：编译器像一位翻译官，把C++代码变成计算机能读懂的0和1密码。  
3. **计算机执行**：计算机拿到密码后，立刻按照指令工作！  

举个🌰：  
```cpp  
#include <iostream>  
int main() {  
    std::cout << "你好！"; 
    return 0;  
}  
```  
编译器会把这段代码翻译成类似：  
`11001010 10110011...`（一堆0和1），计算机看到后就会在屏幕上显示“你好！”

## Hello World <Badge type="tip" text="重点" />

在学习任何编程语言时，第一个程序通常都是打印出“Hello World!”。这是编程世界的“敲门砖”，就像你第一次拿起笔写下自己的名字一样。今天，我们就来一起探索如何在 C++ 中实现这个经典的程序，开启你的编程之旅！

::: code-group

```cpp [编写主函数]
int main() {  // 程序的入口

    // 需要执行的代码写在这里

    return 0; // 程序的出口
}
```

```cpp [引入工具箱]
#include <iostream>  // 引入一个工具箱 [!code highlight]

int main() {  

    // 使用工具箱里的工具， cout 和 endl 
    std::cout << "Hello World!" << std::endl; // [!code highlight]

    return 0; 
}
```

```cpp [简化]
#include <iostream>

// 工具有很多，具体用哪一个工具？
// 这里告诉计算机，默认使用带std标签的工具
using namespace std; // [!code highlight]

int main() {  

    // 现在不用写 std:: 了，是不是更方便了
    cout << "Hello World!" << endl;  //  [!code highlight]

    return 0; 
}
```
:::

程序执行屏幕会输出

<RunningResult>
Hello World!
</RunningResult>

#### 代码解释

1. `#include <iostream>`：这行代码告诉编译器我们要使用 `iostream` 库，它包含了输入和输出的功能。就像你做饭时需要锅和铲子一样，写程序也需要一些工具。
2. `using namespace std;`：这行代码让我们可以直接使用 `std` 命名空间中的函数和对象，比如 `cout`。
3. `int main()`：这是程序的主函数，每个 C++ 程序都必须有一个 `main` 函数，程序从这里开始执行。就像每个故事都有一个开头一样。
4. `cout << "Hello World!" << endl;`：这行代码使用 `cout` 来输出`“Hello World!”`，`endl` 表示换行。`cout` 就像一台打印机，而 `endl` 是它的“换行键”。
5. `return 0;`：这行代码表示程序正常结束。就像你完成作业后写上“完成”一样，告诉老师任务已经完成了。

#### 官方术语
> 术语，同学们需要记忆一下噢
```cpp
#include <iostream> // 引入一个头文件（标准输入输出库）

using namespace std; // 使用std命名空间（标准命名空间）

int main() {  // 主函数，也称为main函数

    cout << "Hello World!" << endl;  // 普通的C++语句

    return 0; // 返回值
}
```

## 两大类语句
在C++中，代码可以分为两大类：`​预处理器指令` 和 `普通语句`

**​预处理器指令**：`以 # 开头`，在编译前处理，用于控制编译过程或引入资源，`不需要分号结束`<br/>
**​普通语句**：普通语句是C++程序的主体部分，是程序的实际执行逻辑。`必须以分号结束`
::: code-group
```cpp [预处理器指令]
#include <iostream> // [!code highlight]

using namespace std; 
int main() {  
    cout << "Hello World!" << endl;  
    return 0; 
}
```

```cpp [普通语句]
#include <iostream> 

using namespace std;  // [!code highlight]

int main() {  
    cout << "Hello World!" << endl;   // [!code highlight]
    return 0;  // [!code highlight]
}
```
:::
## `cout` 语句 <Badge type="tip" text="重点" />

`cout` 是 C++ 中用来输出内容到屏幕的语句。它的特别之处在于它支持**链式调用**，这意味着你可以用多个 `<<` 运算符将多个内容连在一起输出，就像用胶水把东西粘在一起一样！

#### 基本格式：

```cpp
cout << "你要输出的内容" << endl;
```

- `<<`：这是插入运算符，用来将内容插入到 `cout` 中。
- `"你要输出的内容"`：这是你要输出的文本，需要用双引号括起来。
- `endl`：表示换行，相当于按下了回车键。

#### 链式调用示例

```cpp
cout << "我的名字是小明。" << " 我今年" << 12 << "岁。" << endl;
```

输出结果：

<RunningResult>
我的名字是小明。 我今年12岁。
</RunningResult>

#### 换行与不换行

- **使用 `endl`**：输出内容后会换行。
```cpp
cout << "这是第一行。" << endl;
cout << "这是第二行。" << endl;
```
输出结果：
<RunningResult>
这是第一行。<br/>
这是第二行。
</RunningResult>

- **不使用 `endl`**：输出内容会连在一起，不会换行。
```cpp
cout << "这是第一行。";
cout << "这是第二行。";
```
输出结果：
<RunningResult>
这是第一行。这是第二行。
</RunningResult>

## 输出表达式 <Badge type="info" text="这节课先了解" />

在 C++ 中，我们可以使用 `cout` 输出数学表达式的结果。C++ 支持基本的数学运算符，包括加（`+`）、减（`-`）、乘（`*`）、除（`/`）。
注意表达式不需要添加双引号

```cpp
cout << 10 + 5 << endl;
cout << 10 - 5 << endl;
cout << 10 * 5 << endl;
cout << 10 / 5 << endl;
```

输出结果：
<RunningResult>
15<br/>
5<br/>
50<br/>
2<br/>
</RunningResult>

## 注释的使用 <Badge type="tip" text="重点" /> 

在编写代码时，我们经常会添加一些注释，用来解释代码的作用。注释不会被编译器执行，它只是给我们自己或其他人看的。

C++ 中有两种注释方式：

#### 单行注释

使用 `//` 来添加单行注释：

```cpp
// 这是一个单行注释
cout << "Hello World!" << endl; // 这行代码输出 Hello World!
```

#### 多行注释

使用 `/*` 和 `*/` 来添加多行注释：

```cpp
/*
这是一个多行注释。
它可以跨越多行。
*/
cout << "Hello World!" << endl;
```
