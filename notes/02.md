## 什么是变量 <Badge type="tip" text="重点" />

变量是编程中用来存储数据的容器。你可以把变量想象成一个**盒子**，里面可以存放各种类型的数据，比如数字、文字等。每个变量都有一个**名字**（变量名），通过这个名字，我们可以访问或修改盒子里的数据。

**不同盒子，可以存储的数据不同**

在C++中，变量的类型决定了这个“盒子”可以存储什么样的数据。就像现实生活中的盒子有不同的用途一样，变量也有不同的类型，每种类型对应不同的数据。

**改写之前的Hello World程序，把数据存在变量里**

```cpp
#include <iostream>
using namespace std;

int main() {  

    // 把返回值和Hello World都存在盒子里
    int number = 0;  
    string text = "Hello World!"; 

    // 通过变量名，使用盒子里的数据
    cout << text << endl;
    return number;

}
```

## 计算机里的数据 <Badge type="info" text="这节课先了解" />

在C++中，数据有不同的类型，常见的数据类型包括 **整型** 、**浮点型**、**字符型** 和 **字符串** 。这些数据类型是编程的基础，它们决定了变量可以存储的数据种类和范围。这节课我们先认识这些数据类型，为后续的学习打下坚实的基础。

| 数据类型  | 声明关键字       | 声明变量        | 可存储内容                   |
| --------- | ---------------- | --------------- | ---------------------------- |
| 整型      | `int`            | `int a;`        | `15`, `0`, `-9`, `1`         |
| 浮点型    | `double`         | `double b, c;`  | `-8.9`, `15.4`, `0.1`, `1.0` |
| 字符型    | `char`           | `char d;`       | `'a'`, `'l'`, `'1'`          |
| 字符串    | `string`         | `string e;`     | `"abc"`, `"12@#"`, `"1"`     |


- **`int`**：整数类型，用于存储`整数`
- **`double`**：双精度浮点数类型，用于存储`小数`
- **`char`**：字符类型，用于存储`单个字符`
- **`string`**：字符串类型，用于存储`一串字符`

## 如何使用变量 <Badge type="tip" text="重点" />

在C++中，变量需要先**定义**才能使用。定义变量时，需要指定变量的**类型**和**名字**，同时可以选择给它赋一个初始值（称为**初始化**）

**语法：** `数据类型 变量名 = 初始值;`

### 变量的定义
定义一个变量，包含 `声明` 和 `赋值` 两部分，第一次赋值，称为`变量的初始化`
::: code-group
```cpp [方式一]
// 声明并赋值
int age = 12;
double height = 1.65;
char grade = 'A';
string name = "Alice";

```

```cpp [方式二]
// 先声明
int age;
double height;

// 再赋值（第一次赋值称为初始化）
age = 12;
height = 1.65;
```

```cpp [方式三]
// 如果有多个同类型的变量，可以一起定义
int x, y, z;

// 赋值（第一次赋值称为初始化）
x = 10;
y = 20;
z = 30;

// 同时定义多个同类型变量（声明和赋值）
int a = 12 , b = 23;
```
:::
**注意：** 变量在定义时可以不初始化，但建议养成初始化的好习惯，避免使用未初始化的变量导致程序出错。

### 赋值运算符

`=` 称为赋值运算符， 变量的值可以通过赋值语句修改

```cpp
int age = 13; 

age = 15;      // 将变量 age 的值修改为 15
cout << age << endl;   // 这里输出15

age = 20;      // 将变量 age 的值修改为 20
cout << age << endl;  // 这里输出20
```

### 变量命名规则

- 变量名只能包含`字母`、`数字` 、`美元符号$` 和 `下划线_`
- 变量名不能以数字开头
- 变量名不能是C++的关键字（如 `int`, `return`, `if` 等）

::: code-group
```cpp [错误]
// 观察以下变量命名

string #@12df = "你好"; // 错误：变量名只能包含字母、数字、美元符号和下划线 [!code error]
string str123123  = "大吉大利，今晚吃鸡";
double num_x = 23.23;
int 1age = 12;            // 错误：变量名不能以数字开头 [!code error]
int return = 5;           // 错误：变量名不能是关键字  [!code error]
char __hello__ = 'a';
int int = 5;              // 错误：变量名不能是关键字  [!code error]
```

```cpp [区分大小写]
// 变量名区分大小写，age 和 Age 是两个不同的变量。

int age = 12;
int Age = 29;
```

```cpp [命名建议]
// 变量名应具有描述性，能够清晰地表达变量的用途（见名知义）

/**
建议使用英语单词当作变量名
如果变量名由多个单词组成
可以使用下划线或驼峰命名法（如 student_age 或 studentAge）
*/
int age = 12;   // 年龄
string name = "张三";  // 名字

// 如不认识单词，可以使用拼音做变量名
int nian_ling = 18 
string ming_zi = "李四"

// 不建议使用单个字母当作变量名
int a = 20
string b = "王五"
```
:::
#### 综合示例
```cpp
#include <iostream>
using namespace std;

int main() {
    int score = 90;
    double price = 9.99;

    cout << "初始分数: " << score << endl;
    cout << "初始价格: " << price << endl;

    score = 95;       // 修改 score 的值
    price = 12.50;    // 修改 price 的值

    cout << "修改后的分数: " << score << endl;
    cout << "修改后的价格: " << price << endl;

    return 0;
}
```

## 算术运算符 <Badge type="tip" text="重点" />

C++中常用的算术运算符包括：
`+` 加法
`-` 减法
`*` 乘法
`/` 除法
`%` 取模（求余数）

```cpp
int a = 10;
int b = 3;
int sum = a + b;            // sum 的值为 13
int difference = a - b;     // difference 的值为 7
int product = a * b;        // product 的值为 30
int quotient = a / b;       // quotient 的值为 3
int remainder = a % b;      // remainder 的值为 1
```

#### 取模运算符 `%`

取模运算符 `%` 用于计算两个整数相除后的**余数**。它是C++中常用的算术运算符之一，尤其在处理循环、条件判断和数学问题时非常有用。

```cpp
int remainder = 10 % 4; // remainder 的值为 1
```

在这个例子中，`10` 除以 `4` 的商是 `2`，余数是 `2`，所以 `10 % 4` 的结果是 `2`。

**注意事项：**
- 整数相除时，结果会舍去小数部分。例如，`10 / 3` 的结果是 `3`。
- 取模运算符 `%` 只能用于整数类型。


## `cin` 的使用 <Badge type="tip" text="重点" />

`cin` 是C++中用于从键盘获取用户输入的工具。与 `cout` 类似，`cin` 也属于 `std` 命名空间，并且支持**链式调用**。使用 `cin` 时，需要先定义一个变量来存储用户输入的数据。

#### 基本格式

```cpp
cin >> 变量名;
```

- `cin` 会等待用户输入数据，直到用户按下回车键。
- 用户输入的数据类型必须与变量类型匹配，否则可能会导致程序出错。
- 如果用户输入了非法的数据（例如输入字母而不是数字），程序可能会进入错误状态。

#### 示例代码

```cpp
#include <iostream>
using namespace std;

int main() {
    int num;
    cin >> num; // 从键盘获取用户输入，并存入 num 变量中 [!code highlight]
    cout << num + 100 << endl; // 输出 num + 100的值
    return 0;
}
```

#### 程序运行
<RunningResult>
3100 (这里为用户输入)<br/>
3200 <br/>
</RunningResult>

#### 链式调用

`cin` 支持链式调用，这意味着可以在一行代码中获取多个输入。

```cpp
#include <iostream>
using namespace std;

int main() {
    int a, b, c;
    cin >> a >> b >> c; // 从键盘获取用户输入，并分别存入 a,b,c 变量中 [!code highlight]
    cout << a + b + c << endl; // 输出a+b+c的值
    return 0;
}
```

#### 程序运行
> cin 可以连续从键盘读取想要的数据，以`空格`、`tab` 或`换行`作为分隔符。
<RunningResult>

</RunningResult>
