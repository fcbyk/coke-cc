<QA question="什么是布尔类型？如何使用它？" level="tip">

布尔类型是编程中用来表示"真"或"假"的数据类型，就像一个开关🎮，只有两种状态：

- `true`：表示"真"
- `false`：表示"假"

**声明布尔变量的方法**：
```cpp
bool 变量名 = true;  // ✅ 正确写法  
bool 变量名 = false; // ✅ 另一种正确写法
bool 变量名 = 123;   // ❌ 错误！布尔值只能是true或false
```

布尔变量就像是判断题的答案✏️，只能选择对✅或错❌！

**输出布尔值**：
在C++中，布尔值默认以数字形式输出：`true`会输出为`1`，`false`会输出为`0`。如果想以文本形式输出，可以使用`boolalpha`操纵符：

```cpp
#include <iostream>  
using namespace std;  

int main() {  
    bool is_sunny = true;  
    cout << "今天晴天吗？" << is_sunny << endl;  // 输出1
    
    cout << boolalpha; // 设置布尔值以文本形式输出
    cout << "今天晴天吗？" << is_sunny << endl;  // 输出true
    return 0;  
}  
```

运行结果：
```
今天晴天吗？1
今天晴天吗？true
```

这里有个有趣的发现：在计算机内部，`1`代表"真"，`0`代表"假"。

<QA question="关系运算符是什么？有哪些常用的关系运算符？" level="tip">

关系运算符用于比较两个值之间的关系，返回一个布尔值（`true`或`false`）。与算术运算符不同，关系运算符的结果是布尔类型。

**算术运算符vs关系运算符**：

| **类型** | **符号例子** | **作用** | **结果类型** |
| --- | --- | --- | --- |
| **算术运算符** | `+`, `-`, `*`, `/` | 数学计算 | **数字**（如10） |
| **关系运算符** | `==`, `>`, `<=` | **比较大小** | **布尔值**（`true`/`false`） |

你可以把算术运算符想象成计算器（帮你算数：3 + 5 = 8），而关系运算符像裁判（判断对错：3 > 5？答案是`false`！）

**常用关系运算符**：

假设有两个变量，a=10，b=20

| **运算符** | **含义** | **例子** | **结果** |
| --- | --- | --- | --- |
| `==` | 等于 | `a == b` | `false`（10不等于20） |
| `!=` | 不等于 | `a != b` | `true`（10确实不等于20） |
| `>` | 大于 | `a > b` | `false`（10不大于20） |
| `<` | 小于 | `a < b` | `true`（10小于20） |
| `>=` | 大于等于 | `a >= 10` | `true`（10等于10） |
| `<=` | 小于等于 | `b <= 25` | `true`（20小于25） |

**使用示例**：

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 100;
    int b = 200;

    cout << (a > b);  //输出0（假）
    cout << (a < b);  //输出1（真）
    
    cout << (a >= b); //输出0（假）
    cout << (a <= b); //输出1（真）
    
    cout << (a == b); //输出0（假）
    cout << (a != b); //输出1（真）
    return 0;
}
```

运行结果：
```
010101
```

**注意**：使用关系运算符时，为避免与`cout`的`<<`运算符冲突，我们需要用括号将表达式括起来。

<QA question="什么是逻辑运算符？如何在代码中使用它们？" level="tip">

逻辑运算符用于组合多个条件，将多个布尔值（`true`/`false`）组合成一个最终的布尔值。它们就像是条件的"组合器"🔗。

**三种主要的逻辑运算符**：

| **运算符** | **名字** | **作用** | **例子** | **结果** |
| --- | --- | --- | --- | --- |
| `&&` | 与（AND） | **所有条件都为真 → 才返回**`true` | `true && true` | `true` |
| `\|\|` | 或（OR） | **至少一个条件为真 → 就返回**`true` | `false \|\| true` | `true` |
| `!` | 非（NOT） | **反转布尔值**（真变假，假变真） | `!true` | `false` |

**生活中的"与"运算例子**：
1. 如果你写完作业**并且**打扫完房间，你就可以吃冰淇淋
   - 意味着`写完作业`和`打扫完房间`两个条件都必须满足
2. 如果今天是周末**而且**天气晴朗，我们就去公园玩
   - 必须`是周末`且`天气晴朗`两个条件都成立

**生活中的"或"运算例子**：
1. 中秋节**或者**国庆节，我们就放假
   - 只要满足`是中秋节`或`是国庆节`任一条件即可
2. 你选择项羽**或者**程咬金，我们的队伍阵容就完美了
   - 只要满足`选择项羽`或`选择程咬金`任一条件即可

**代码示例**：

```cpp
#include <iostream>
using namespace std;

int main() {
    // 使用变量存储学生成绩
    int math = 92;    // 数学成绩
    int english = 70; // 英语成绩

    // 使用与运算判断两个科目都及格吗
    bool is_pass = math >= 60 && english >= 60;
    cout << is_pass << endl;    // 输出1（真）

    // 使用或运算判断是否有科目超过90分
    bool is_good = math > 90 || english > 90;
    cout << is_good << endl;    // 输出1（真）
    
    return 0;
}
```

运行结果：
```
1
1
```

这个例子中：
- `is_pass`为`true`，因为数学和英语都超过60分
- `is_good`为`true`，因为数学超过90分（虽然英语没有）

<QA question="if条件语句如何使用？什么是else if？" level="tip">

`if`语句是根据条件决定是否执行某段代码的结构。你可以把它想象成生活中的决策：
- 如果下雨，就带伞；否则，戴帽子
- 如果作业写完，就能玩游戏；否则，继续写作业

**基本结构**：

```cpp
if (条件) {  
    // 条件为真时执行的代码  
} else {  
    // 条件为假时执行的代码  
}  
```

**使用规则**：
1. 条件必须是布尔值（`true`/`false`）或能得到布尔值的表达式
2. 代码块用大括号`{}`包裹，缩进使代码更易读

**示例1：判断成绩是否及格**

```cpp
#include <iostream>  
using namespace std;  

int main() {  
    int score = 85;  
    if (score >= 60) {  
        cout << "🎉 恭喜，及格啦！" << endl;  
    } else {  
        cout << "😢 要加油哦！" << endl;  
    }  
    return 0;  
}  
```

运行结果：
```
🎉 恭喜，及格啦！
```

**示例2：根据天气选择活动**

```cpp
bool is_raining = true;  
if (is_raining) {  
    cout << "在家玩桌游吧！♟️" << endl;  
} else {  
    cout << "去公园踢足球！⚽" << endl;  
}  
```

运行结果：
```
在家玩桌游吧！♟️
```

**多条件判断：else if**

当需要判断多个条件时，可以使用`else if`连接多个条件判断。程序会从上到下检查条件，一旦找到第一个满足的条件，就执行对应的代码块，然后跳过其余的判断。

**示例：根据成绩判断等级**

```cpp
int score = 85;  
if (score >= 90) {  
    cout << "🌟 优秀！" << endl;  
} else if (score >= 80) {  
    cout << "👍 良好！" << endl;  
} else if (score >= 60) {  
    cout << "✅ 及格！" << endl;  
} else {  
    cout << "💪 继续努力！" << endl;  
}  
```

运行结果：
```
👍 良好！
```

**执行顺序**：
1. 先检查`score >= 90`，结果为`false`（85不大于等于90）
2. 再检查`score >= 80`，结果为`true`（85大于等于80）
3. 执行对应代码块，输出"👍 良好！"
4. 跳过后续所有条件判断
