<QA question="什么是转义字符？常用的转义字符有哪些？" level="tip">

转义字符是C++中用于表示特殊字符的一种方式，以反斜杠 (`\`) 开头，后面跟一个特定的字符。转义字符的作用可以分为两种：

1. **将特殊字符转义为普通字符**：有些字符在C++中有特殊含义（如双引号 `"`、反斜杠 `\` 等），通过转义字符可以将它们转义为普通字符。

2. **将普通字符转义为特殊字符**：有些字符本身是普通字符（如 `n`、`t` 等），通过转义字符可以将它们转换为具有特殊功能的字符。

**常用的转义字符包括：**

| 转义字符 | 描述 |
| :--: | :--: |
| `\n` | 换行符 |
| `\t` | 水平制表符 (Tab) |
| `\\` | 反斜杠 |
| `\"` | 双引号 |
| `\'` | 单引号 |
| `\0` | 空字符 (Null) |

下面是一些使用转义字符的例子：

```cpp
#include <iostream>
using namespace std;

int main(){
    // 输出双引号
    cout << "丁真:\"你好，我是丁真\"" << endl;

    // 换行符
    cout << "第一行\n第二行\n";

    // 输出反斜杠
    cout << "\\" << endl;

    // 结束符，\0 表示字符串结束
    cout << "你好，\0世界" << endl;

    return 0;
}
```

运行结果：
```
丁真:"你好，我是丁真"
第一行
第二行
\
你好，
```

<QA question="C++中的算术运算符有哪些？它们的作用是什么？" level="tip">

C++中的算术运算符用于执行基本的数学运算，如加法、减法、乘法和除法。以下是常用的算术运算符及其用法：

| 运算符 | 描述 | 示例 |
| :--: | :--: | :--: |
| `+` | 加法 | `a + b` |
| `-` | 减法 | `a - b` |
| `*` | 乘法 | `a * b` |
| `/` | 除法 | `a / b` |
| `%` | 取模（取余数） | `a % b` |
| `++` | 自增（加 1） | `a++` 或 `++a` |
| `--` | 自减（减 1） | `a--` 或 `--a` |

**1. 加法 (`+`)**：将两个数相加
```cpp
int a = 5, b = 3;
int sum = a + b; // sum = 8
```

**2. 减法 (`-`)**：将两个数相减
```cpp
int a = 5, b = 3;
int difference = a - b; // difference = 2
```

**3. 乘法 (`*`)**：将两个数相乘
```cpp
int a = 5, b = 3;
int product = a * b; // product = 15
```

**4. 除法 (`/`)**：将两个数相除。注意整数除法会舍去小数部分
```cpp
int a = 5, b = 2;
int quotient = a / b; // quotient = 2（整数除法）

double c = 5.0, d = 2.0;
double result = c / d; // result = 2.5（浮点数除法）
```

**5. 取模 (`%`)**：返回两个整数相除的余数
```cpp
int a = 5, b = 2;
int remainder = a % b; // remainder = 1
```

**6. 自增 (`++`)**：将变量的值加1。分为前缀自增和后缀自增：
- 前缀自增 (`++a`)：先加1，再使用变量的值
- 后缀自增 (`a++`)：先使用变量的值，再加1
```cpp
int a = 5;
int b = ++a; // a = 6, b = 6（先加1，再赋值）

int c = 5;
int d = c++; // c = 6, d = 5（先赋值，再加1）
```

**7. 自减 (`--`)**：将变量的值减1。分为前缀自减和后缀自减：
- 前缀自减 (`--a`)：先减1，再使用变量的值
- 后缀自减 (`a--`)：先使用变量的值，再减1
```cpp
int a = 5;
int b = --a; // a = 4, b = 4（先减1，再赋值）

int c = 5;
int d = c--; // c = 4, d = 5（先赋值，再减1）
```

下面是一个综合示例：

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 3;
    cout << "a + b = " << (a + b) << endl;
    cout << "a - b = " << (a - b) << endl;
    cout << "a * b = " << (a * b) << endl;
    cout << "a / b = " << (a / b) << endl;
    cout << "a % b = " << (a % b) << endl;

    int c = 5;
    cout << "c++ = " << c++ << endl;
    cout << "c = " << c << endl;

    int d = 5;
    cout << "--d = " << --d << endl;
    cout << "d = " << d << endl;

    return 0;
}
```

运行结果：
```
a + b = 13
a - b = 7
a * b = 30
a / b = 3
a % b = 1
c++ = 5
c = 6
--d = 4
d = 4
```

<QA question="如何使用整除和取余运算符提取数字的各个位数？" level="tip">

在C++中，整除（`/`）和取余（`%`）运算符不仅可以用于基本的数学计算，还可以用于提取数字的各个位数。这是处理整数的一个非常实用的技巧。

**整除运算符（`/`）**：
- 返回两个整数相除的商（舍去小数部分）
- 例如：`10 / 3 = 3`

**取余运算符（`%`）**：
- 返回两个整数相除的余数
- 例如：`10 % 3 = 1`

**提取数字位数的方法：**

1. **提取个位数**：对原数字取余10（`number % 10`）
2. **提取十位数**：先整除10，再取余10（`(number / 10) % 10`）
3. **提取百位数**：先整除100，再取余10（`(number / 100) % 10`）
4. **提取千位数**：先整除1000，再取余10（`(number / 1000) % 10`）
5. **以此类推...**

下面是一个示例，演示如何提取一个三位数的各个位数：

```cpp
#include <iostream>
using namespace std;

int main() {
    int number = 123;

    // 提取个位数
    int units = number % 10; // 3

    // 提取十位数
    int tens = (number / 10) % 10; // 2

    // 提取百位数
    int hundreds = (number / 100) % 10; // 1

    cout << "当前数字: " << number << endl;
    cout << "百位数: " << hundreds << endl;
    cout << "十位数: " << tens << endl;
    cout << "个位数: " << units << endl;

    return 0;
}
```

运行结果：
```
当前数字: 123
百位数: 1
十位数: 2
个位数: 3
```

对于四位数或五位数，可以依照同样的规律提取：

```cpp
int number = 12345;

// 提取个位数
int units = number % 10; // 5

// 提取十位数
int tens = (number / 10) % 10; // 4

// 提取百位数
int hundreds = (number / 100) % 10; // 3

// 提取千位数
int thousands = (number / 1000) % 10; // 2

// 提取万位数
int tenThousands = (number / 10000) % 10; // 1
```

这种方法的原理是：通过整除操作将目标位数移到个位，然后通过取余操作提取个位数字。

<QA question="if语句是什么？它的基本语法是什么？" level="info">

`if`语句（也称为分支语句）是一种控制流语句，用于根据条件是否为真（非零）或假（零）来决定执行哪段代码。

**基本语法：**

```cpp
if (条件) {
    // 条件为真（非零）时执行的代码
} else {
    // 条件为假（零）时执行的代码
}
```

- **条件**：一个表达式，结果为`0`（假）或非`0`（真）。
- **代码块**：根据条件的真假执行对应的代码。

在C++中，任何非零值都被视为真，而零被视为假。这意味着你可以直接使用数字作为条件：

```cpp
#include <iostream>
using namespace std;

int main() {
    int num = 0;
    if (num) { // num为零，条件为假
        cout << "num的值不为0" << endl;
    } else {
        cout << "num的值为0" << endl;
    }
    return 0;
}
```

运行结果：
```
num的值为0
```

**注意：**
- 如果条件后只有一条语句，可以省略大括号`{}`，但为了代码的可读性和避免潜在错误，建议始终使用大括号。
